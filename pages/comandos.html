Comandos sem Flow

Comando Resultado
git add Move alterações para a área de staging
git branch Cria e gerencia branches
git checkout Utilizado para navegar entre branches ou arquivos e commits
git clean Remove todos os arquivos não rastreados pelo Git
git clone Cria uma cópia de um repositório
git commit Envia as alterações na area de staging para o histórico do projeto
git commit --amend Permite mergir novas alterações ao commit mais recente
git config Permite alterara configurações da instalação do Git
git fetch Baixa a branch de outro repositório com todos os commits associados, mas sem integrar com o repositório atual
git init Inicializa um novo repositório
git log Exibe commits do projeto
git merge Mescla o branch especificado com o branch atual
git pull Utilizado para baixar a branch e mesclar automaticamente
git push Permite publicar mudanças em um branch
git rebase Permite mover branches para evitar mesclagens desnecessárias
git rebase -i A flag -i permite realizar mudanças como adicionar, editar e deletar commits
git reflog Permite voltar para alterações que não estão rastreadas
git remote Comando para administrar coneções remotas
git reset Permite desfazer alterações no diretório local
git revert Desfaz um commit
git status Exibe alterações feitas no projeto
